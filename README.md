- Endpoint crear usuarios (/createuser) - Metodo: post

- Endpoint modificar usuario (/modifyuser) - Metodo: update

- Endpoint eliminar usuario (/deleteuser) - Metodo: delete

- Endpoint muestra informacion (/userinfo) - Metodo get
